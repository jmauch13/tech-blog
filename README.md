# tech-blog
## Purpose
The purpose of this app is to create a tech blog with posts to read and an area for users to add.  It includes a homepage to sign up and to log in.
## Technologies used
This app is made with:
* JavaScript
* Handlebars
* Express
* Express-session
* mysql2
* bcrypt
* dotenv
* Sequelize
